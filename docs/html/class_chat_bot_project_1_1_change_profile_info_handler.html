Detailed Description

Member Enumeration Documentation

◆ ChangeProfileInfoState

enum ChangeProfileInfoState

Estados por los que pasara el handler para asi saber que mensaje esperar y que respuesta dar.

Enumerator
Start 

-Start: Es el estadio inicial del comando.

AwaitingWhatToChange 

-AwaitingWhatToChange: En este estado el mensaje esperado es un /Name o /Password, para saber a que estado continuar.

ChangeName 

-ChangeName: En este estado el mensaje esperado es el nuevo nombre que el usuario quiere utilizar.

ChangePassword 

-ChangePassword: En este estado el mensaje esperado es la nueva contraseña que el usuario quiere utilizar.
