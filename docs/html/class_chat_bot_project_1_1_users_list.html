Detailed Description

UsersList es el experto en conocer a todos los usuarios, y por el patron Expert este tambien es quien posee la responsabilidad de agregar y/o remover Usuario. Debido a que UsersList contiene instancias de User, por Creator, esta clase debe tener la responsabilidad de crear dichas instancias.
